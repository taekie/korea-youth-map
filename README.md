# 대한민국 시군구 청년 성별격차 지도

250개 시군구의 인구 지표를 인터랙티브 코로플레스 지도로 보여줍니다.

**라이브:** https://taekie.github.io/korea-youth-map/

## 지표
- **0–9세 / 10–19세 인구비율** — 유소년 비중
- **청년 성별격차 (20–39세)** — 남성 점유율 − 여성 점유율 (%p)
- **전체 성별격차**
- **고령화율 (65세+)**, **중위연령**

성별격차 지표는 `전국평균 기준` 체크 시 컬러 중립점이 전국평균으로 이동해 "전국 대비 높고 낮음"을 볼 수 있습니다. 지역 클릭 시 인구 피라미드(전국 기준선 대비)가 열립니다.

## 데이터
- 행정안전부 주민등록인구 (2026년 6월, 5세 연령구간 · 성별)
- 경계: 통계청 시군구 2018 (250개). 이후 변경(군위군 대구 편입, 인천 남구→미추홀구)은 반영해 매칭.

## 기술
- D3.js v7 (지도·스케일·피라미드), TopoJSON (시도 경계 mesh)
- 정적 사이트 (빌드 불필요)
